# mini_project

🍽️ Hostel Food Waste Prediction using Machine Learning

A Data Science mini-project that analyzes and predicts food waste in hostel dining facilities using Python, Machine Learning, and Data Visualization.
This project demonstrates how AI-driven insights can help reduce food waste, cut costs, and promote sustainability in hostel management.

📌 Project Overview

Food waste in hostels is a major economic and environmental issue. Every day, meals are prepared in bulk, but not all are consumed.
This project:
📊 Analyzes hostel meal data
📈 Visualizes preparation vs consumption patterns
🤖 Builds a Machine Learning model to predict food waste
💡 Suggests data-driven waste reduction strategies

📂 Dataset Description

The dataset contains hostel meal records with the following attributes:
Prepared – Number of meals cooked
Consumed – Number of meals eaten
Wasted – Meals wasted (Target Variable)
Strategy – Waste-reduction strategy applied
Meal – Breakfast / Lunch / Dinner

The dataset includes both numerical and categorical features.

🧹 Data Preprocessing

Removed duplicate records
Handled missing values
Converted categorical features into numeric values
Breakfast → 0
Lunch → 1
Dinner → 2
Awareness Campaign → 1
Smaller Portions → 2
Donation → 3

Preprocessing ensures compatibility with machine learning algorithms.

🛠️ Technologies Used

Python
Pandas

Scikit-learn
Power BI (Dashboard Visualization)

📄 Conclusion

This project proves that data-driven decision-making can significantly reduce food waste. By combining visualization and machine learning, hostel management can plan meals efficiently and promote sustainability.
